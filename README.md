#Project Overview

This project showcases an interactive Sales Dashboard built in Power BI to analyze business performance across product categories, products, customers, and time periods. The dashboard helps track key sales KPIs and provides actionable insights for decision-making.

#The report highlights:

Total Sales: $102.53K
Total Orders: 100
Total Customers: 12
Top Product Category: Laptop ($52K)

#Tools & Technologies Used:

Power BI Desktop
Power Query for data cleaning and transformation
DAX for KPI calculations
Data Modeling for relationships
Interactive Visualizations

#Dashboard Development Workflow:

Requirement gathering and KPI planning
Data loading from source files
Data cleaning using Power Query
Creating DAX measures for KPIs
Data modeling and relationships
Building charts, cards, and slicers
Testing and validating metrics
